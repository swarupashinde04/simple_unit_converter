# Simple Unit Converter

A simple web application built using **Python and Flask** that converts basic units such as kilometers to meters, kilograms to grams, and Celsius to Fahrenheit.

This project was created as part of my learning journey while exploring **Python web development with Flask**.

---

## Features

- Convert **Kilometers → Meters**
- Convert **Kilograms → Grams**
- Convert **Celsius → Fahrenheit**
- Simple and clean user interface
- Fast conversion using Flask backend

---

- Flask

## How to Run the Project

1. Clone the repository


git clone https://github.com/yourusername/simple-unit-converter.git


2. Navigate to the project folder


cd simple-unit-converter


3. Install Flask


pip install flask


4. Run the application


python app.py


5. Open the browser and go to


http://127.0.0.1:5000
